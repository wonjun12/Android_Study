SharedPreference
 - 안드로이드에서 파일로 저장하는 공간

RecyclerView
 - 리스트의 틀을 만드는 레이아웃
 - 안의 리스트를 보여주기 위해서는 새로 레이아웃을 만들어야 한다.
   - Layout Resource File 만들기
       - memo_row (이름 지정)
       - root Element => LinearLayout 안에 만들어질꺼기 때문에 지정하자.
     - 리스트의 하나의 행을 만들기 위해 만든것이다.



= CardView
  - 카드 처럼 View를 보여주는 레이아웃
    - cardElevation : 그림자
    - radio : 모서리 각진 정도

  - 데이터를 담을 클래스를 만들어야한다.
   -> 데이터를 담은 클래스와 데이터를 보여주는 레이아웃을 연결시켜주는 클래스를 만들어야한다.
